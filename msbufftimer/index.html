Click on a timer to start it.
Create custom timers by selecting an icon and sound.
Can use links to audio files for additional sounds.
Countdown timer starts a timer with a duration of mm minutes ss seconds.
Click the timer to remove it.
Tooltip memo adds a tooltip when the timer is hovered over and is also the note displayed when the timer is up.
Loop will make the timer loop. This is indicated with a green border. Alarm sets a timer for the next mm:ss on the hour.
Use for atmospheric exp buffs at set times.
Snooze on click makes it so the alarm repeats until the timer is clicked on.
This is indicated with an orange border.
If Save is checked, custom timers will be added to a preset.
Toggle Remove button to remove from preset and Use button to use the timer.
